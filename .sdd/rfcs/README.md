# RFCs And ADRs

This folder stores architecture decisions and request-for-comment documents.

Use an ADR when a decision affects multiple features, database shape, security,
deployment, or long-term maintainability.

## Naming

```text
ADR-001-architecture.md
ADR-002-database-design.md
ADR-003-authentication-approach.md
```

## Recommended ADR Sections

- Status
- Date
- Context
- Decision
- Alternatives Considered
- Consequences
- Follow-Up Work
