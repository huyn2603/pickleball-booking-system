# Reviews

This folder stores SDD review notes, gap checks, and closeout records.

Use review files when:

- Importing or adapting external SDD material.
- Closing a spec phase.
- Recording open questions.
- Documenting a database/API/security gap.
- Explaining why a seemingly relevant feature was not imported.

## Naming

```text
{topic}-{yyyy-mm-dd}.md
```

Examples:

```text
sdd-import-gap-review-2026-06-23.md
booking-overlap-review-2026-06-23.md
payment-refund-state-review-2026-06-23.md
```

## Recommended Sections

- Source / Scope
- Findings
- Decisions
- Added / Changed Files
- Not Imported / Deferred
- Risks
- Next Steps
